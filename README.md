## Projects

### [Barras Nómades](https://barras-nomades.vercel.app) — Landing page for real client · *in production*
Landing page for a bartending company in Buenos Aires. Built with Next.js 16, TypeScript and Tailwind CSS v4.  
Scroll-reveal animations with native IntersectionObserver, dynamic OG images via `next/og`, full technical SEO (Open Graph, JSON-LD LocalBusiness schema, sitemap.xml, robots.txt), Vercel Analytics.

### [Peto Tatts](https://peto-tatts.vercel.app) — Portfolio for real client · *in production*
Multipage site for a professional tattoo artist in Buenos Aires. Dark visual identity built from scratch.  
Gallery, testimonials, contact form with WhatsApp CTA. End-to-end design and development with no templates.

### [BoletoClick](https://boletoclick-front.vercel.app) — Ticketing platform · *group project, 10 devs*
Full ticketing platform for massive events. Three user profiles: buyer, producer, and admin.  
Stripe payments · Google OAuth · OpenAI/ChatGPT assistant · Mapbox geolocation · automated email crons (newsletter + notices) · QR tickets · discount codes.  
Stack: Next.js · NestJS · TypeScript · Tailwind · Supabase · PostgreSQL · Railway · Vercel.  
Built with feature branches, pull requests and continuous code reviews.

### [Appointment Scheduling System](https://github.com/fermintaboada/appointment-scheduling-system) — Full stack · *bootcamp project*
Full stack app with authentication (register, login, hashed credentials, protected routes).  
Business rules on the backend: business hours validation, 24h advance booking, no weekends, no duplicates.  
Layered architecture (controllers → services → repositories) with TypeORM and entity modeling.  
Email confirmation via Resend API. Distributed deploy: Vercel + Render + Neon (PostgreSQL).
